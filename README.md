# 🎬 CineHub

## 📌 Purpose of the Project

CineHub is a web-based movie explorer application that allows users to discover, search, and explore movies in real-time. The purpose of this project is to demonstrate the use of JavaScript, API integration, and dynamic UI rendering while building an interactive and responsive web application.

---

## 🌐 APIs Used

### TMDB API (Primary)

* Fetch trending movies
* Search movies
* Get movie details (ratings, release date, posters, etc.)

### OMDB API (Secondary)

* Fetch additional information such as box office collection and IMDB ratings

---

## 🚀 Features

* 🔍 Search movies by title
* 🎬 Display trending and upcoming movies
* 📊 Sort movies by rating or release date
* 🎯 Filter movies based on criteria (e.g., rating)
* 🌙 Dark/Light mode toggle
* ❤️ Favorites feature (planned for future enhancement)

---

## 🛠️ Technologies Used

* **HTML** – Structure of the application
* **CSS** – Styling and responsive design
* **JavaScript** – Functionality and interactivity

### JavaScript Concepts:

* Fetch API
* Async/Await
* DOM Manipulation
* Event Handling
* Higher Order Functions (`map`, `filter`, `sort`, `find`)

## ⚙️ Setup and Run Instructions

1. Clone the repository:

```bash
git clone https://github.com/abhradipghosal18/cinehub.git
```

2. Navigate to the project folder:

```bash
cd cinehub
```

3. Add your API keys in the JavaScript file:

```javascript
const TMDB_API_KEY = "your_tmdb_api_key";
const OMDB_API_KEY = "your_omdb_api_key";
```

4. Run the project:

* Open `index.html` in your browser
